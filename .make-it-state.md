## Auth Flow

1. User clicks "Sign in with SSO" → browser navigates to `/api/auth/login`
2. Server redirects to mock-oidc authorization endpoint
3. User picks a test user in mock-oidc user picker
4. mock-oidc redirects back to `/api/auth/callback` with authorization code
5. Server exchanges code for tokens, calls userinfo, looks up user in DB
6. Server signs JWT with role + permissions from DB, sets httpOnly cookie
7. Browser redirected to `/dashboard`

- Database-driven RBAC (4 system roles, 40+ permissions)
- 6 AI Agents: VERA, CARA, DORA, SARA, RITA, MARS
